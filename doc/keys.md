# Keys

Notations
---
Keys can be written in different ways, here are the supported types:
* Key’s name (ie: F4)
  * `<Key>F4</Key>`
* Key’s decimal value (ie: 115)
  * `<Key>115</Key>`
* Key’s hexadecimal value (ie: 0x73)
  * `<Key>0x73</Key>`

Remember you can set multiple keys by adding multiple `<Key>`:  
```XML
<Key>ALT</Key>
<Key>F4</Key>
```
